    SplinePlus Data

    SPData is a data class for the monobehaviour Spline Plus,
    You can get the SPData if you already have your spline plus game object in your scene hierarchy like bellow. 

    SPData sPData;
public void Start(){
sPData =GetComponent<SplinePlus>().SPData;
}

    When using the Spline plus API, this is how to get the SPData object.

    SPData sPData;
public void Start(){
sPData = SplinePlusAPI.SplinePlus_Create(new Vector3(0,0,0));
}
